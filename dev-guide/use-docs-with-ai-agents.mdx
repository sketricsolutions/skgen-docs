Use this page when you want an AI coding agent or assistant to answer SketricGen questions with the current documentation instead of relying on training data.

SketricGen docs are hosted on Mintlify, which provides agent-readable Markdown, documentation indexes, a product skill file, and a hosted search MCP server.

## Agent-Readable Resources

| Resource | URL | Use it for |
| --- | --- | --- |
| Documentation index | `https://docs.sketricgen.ai/llms.txt` | Discover available pages before opening specific docs |
| Full documentation context | `https://docs.sketricgen.ai/llms-full.txt` | Load broad context when your AI tool supports larger files |
| Product skill | `https://docs.sketricgen.ai/skill.md` | Understand SketricGen capabilities, workflows, inputs, and constraints |
| MCP discovery | `https://docs.sketricgen.ai/.well-known/mcp` | Let MCP-compatible tools discover the SketricGen docs search server |
| Search MCP server | `https://docs.sketricgen.ai/mcp` | Search and retrieve SketricGen docs from MCP-compatible tools |

Every public page is also available as Markdown by adding `.md` to the page URL. For example:

```text
https://docs.sketricgen.ai/dev-guide/public-api-usage.md
```

## Recommended Agent Instruction

Copy this instruction into Codex, Claude, Cursor, or another AI agent when you want it to use SketricGen documentation:

```text
Use the current SketricGen documentation at https://docs.sketricgen.ai.
Start with https://docs.sketricgen.ai/llms.txt to discover relevant pages.
Use https://docs.sketricgen.ai/skill.md for product capabilities and constraints.
Prefer docs.sketricgen.ai pages over older pages under www.sketricgen.ai/docs.
When opening a page for context, use the .md version when available.
```

## Notes for MCP Clients

Mintlify exposes SketricGen docs through a hosted MCP server. Use the MCP discovery URL when your tool supports automatic discovery:

```text
https://docs.sketricgen.ai/.well-known/mcp
```

If your tool asks directly for the server URL, use:

```text
https://docs.sketricgen.ai/mcp
```

The MCP server is for searching and retrieving published docs. It does not give agents access to your SketricGen account, private agents, conversations, files, or API keys.
