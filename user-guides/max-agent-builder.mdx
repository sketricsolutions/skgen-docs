Max - Agent Builder is SketricGen's AI assistant for creating AI Workforce Agent Spaces from a conversation.

Max is best used for internal workflows where you want help turning an idea into a main agent, sub agents, roles, tools, and instructions.

## How Max Works

1. Describe what you want to build.
2. Max asks clarifying questions.
3. Max suggests relevant agents and tools.
4. You review the proposed workflow.
5. Max creates the Agent Space.
6. You connect accounts, add Knowledge Bases, test, and refine.

## What Max Can Help With

Max can help:

- Draft agent roles and instructions.
- Suggest tools and connectors.
- Create multi-agent workflow structure.
- Draft when the main agent should call each sub agent.
- Create a starting point faster than a blank Agent Space.

## What You Still Need to Review

Max creates a draft, not a production guarantee. Review:

- Agent instructions
- Tool access
- Connected accounts
- Knowledge Base selection
- Model choices
- Structured outputs if needed
- Test runs and traces

## Brand Agents vs Max

Use the **Brand Agent wizard** when you want a customer-facing website agent. Use **Max** when you want help building an internal AI Workforce workflow.
