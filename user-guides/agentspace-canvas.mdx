Agent Spaces are now part of **AI Workforce**. For the current guide, see [AI Workforce Agent Spaces](/ai-workforce/agent-spaces).

## What Changed

The old unified AgentSpace workflow model has been split:

- **Brand Agents** handle customer-facing widgets, lead capture, and website-trained agents.
- **AI Workforce Agent Spaces** handle internal multi-agent workflows, longer runs, files, artifacts, tools, and team work.

If you are building a public website agent, start with [Create a Brand Agent](/brand-agents/create-brand-agent). If you are building an internal workflow, use [AI Workforce Agent Spaces](/ai-workforce/agent-spaces).
