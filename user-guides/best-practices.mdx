The most important rules:

- Choose Brand Agents for public customer-facing agents.
- Choose AI Workforce for internal long-running work with files, tools, and team workflows.
- Keep the main agent responsible for the user conversation.
- Use sub agents only for focused helper work.
- Give agents only the tools they need.
- Use Knowledge Bases for trusted answers.
- Test with realistic prompts.
- Inspect traces and artifacts before relying on a workflow.
